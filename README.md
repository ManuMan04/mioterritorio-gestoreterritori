# MioTerritorio.it

Simple, fast, and secure territory management for field service with no server-side data storage.

## Features
- **Mobile First**: Optimized for smartphones.
- **Dark Mode**: System-aligned theme support.
- **Privacy**: Data lives in your browser's LocalStorage.
- **Territory Management**: Organize by address and unit.
- **Stats**: Real-time progress tracking.

## Getting Started
1. Download `index.html`.
2. Open with any modern browser.
3. Add to Home Screen for an app-like experience.

## Tech Stack
- HTML5, Tailwind CSS, Alpine.js
- LocalStorage
